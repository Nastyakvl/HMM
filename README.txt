На вход программе задаются 4 параметра:
1. Название текстового файла, в котором содержится начальное распределение.
Первой строкой в этом файле указывается количество состояний (что совпадает с размерностью 
вектора). 
На следующей строке через пробел указываются начальные вероятности. Сами числа пишутся через 
запятую.
Пример:
2
0,6 0,4

2. Название текстового файла, в котором содержится матрица переходных вероятностей.
Первой строке соответствует первая строка матрицы. Второй строке - вторая строка и так далее.

Пример:
0,6 0,4
0,3 0,7

3. Название текстового файла, в котором содержится вся информация о наблюдаемых событиях.
В первой строке указывается кол-во наблюдаемых состояний.
Во второй строке алфавит, из которого выбирается наблюдения.
Далее идет матрица веротяности выбора одного из наблюдений.

Пример:
3
OneIceCream TwoIceCream ThreeIceCream
0,2 0,5
0,4 0,4
0,4 0,1

4. Кол-во желаемых переходов и наблюдений
